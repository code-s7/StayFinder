# StayFinder

StayFinder is a rental and workspace listing platform built with Node.js, Express.js, MongoDB, and EJS.

## Features

- Create, read, update, and delete listings
- Category-wise filtering for room, office, and shop
- Clean responsive UI for project demonstration

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- HTML/CSS/JavaScript

## Setup

1. Install dependencies:

   npm install

2. Seed sample data:

   npm run seed

3. Start app:

   npm start

4. Open in browser:

   http://localhost:3000/listings

## Listing Fields

- title
- description
- price
- category
- imageUrl
- location
